# Vendor Firmware Binaries
Vendor Compiled Firmware Binaries, git repo for convenience

This repository contains vendor compiled firmware binaries provided for reference, archival, or experimental purposes. \
No source code is included, and no modifications, alterations, or enhancements have been made to the binaries.

- All binary files included are derived from:
  - Direct extraction from device flash memory (e.g., NOR, NAND, eMMC, UFS)
  - Extraction from official firmware update packages or system images
